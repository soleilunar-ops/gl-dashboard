# 합성(더미) 데이터 생성 — v2 논리 구조와 계산식

> 중학생도 이해할 수 있게 작성. 모든 수치의 출처 명시.

## 왜 더미데이터를 만들었나?

핫팩은 **겨울에 팔리는 상품**. 예측 모델이 "겨울에 얼마나 팔릴지" 잘 맞추는지 확인하려면 **겨울 시험 데이터**가 필요.

쿠팡 일간종합성과지표는 **2025-04 ~ 2026-04 (1년)**만 있습니다.

```
2025-04 ──── 여름 ──── 가을 ──── ★겨울★ ──── 2026-04
  ↑ 시작                                        ↑ 끝
```

이 1년을 "공부용(학습)"과 "시험용(검증)"으로 나누면:

- 공부에 겨울 → 시험에 겨울 없음
- 시험에 겨울 → 공부에 겨울 없음

**어떻게 나눠도 겨울 시험 불가능.**

해결: **2024년 판매 데이터를 합성 생성**. 2024년 날씨는 기상청 실측이 있으니, "이 날씨면 이만큼 팔렸을 것"을 추정.

## 어떤 순서로 만들었나

```
[Step 1] 2025 실데이터에서 "체감온도 구간별 판매 민감도" 추출
    ↓
[Step 2] 2024 ASOS 실날씨를 지역 가중 평균 → 체감온도 계산
    ↓
[Step 3] 월별 계절 계수 × 체감온도 편차 보정 × 적설 효과 × 급강하 효과 → 2024 합성 판매
    ↓
[Step 4] 합성 판매 × 월별 실측 발주 비율 → 합성 납품
```

---

## Step 1. 체감온도 구간별 민감도 추출

2025년 실데이터에서 **체감온도 구간별 1℃당 판매 변화**를 실측:

| 체감온도 구간 | 1℃당 판매 변화                   |
| ------------- | -------------------------------- |
| -5℃ 이하      | -274개 (포화, 이미 다 샀음)      |
| -5 ~ 5℃       | **-13,114개** (가장 민감한 구간) |
| 5 ~ 10℃       | -5,385개                         |
| 10 ~ 20℃      | -2,870개                         |
| 20℃ 이상      | -129개 (여름, 영향 없음)         |

**비유**: "0~5℃ 구간에서 1도 떨어지면 13,114개 더 팔림"을 54주 실데이터에서 직접 측정.

---

## Step 2. 체감온도 공식 (기상청 JAG/TI 2001)

```
W_C = 13.12 + 0.6215 × T - 11.37 × V^0.16 + 0.3965 × T × V^0.16
```

- `T`: 기온 (℃)
- `V`: 풍속 (km/h, 지상 10m)
- 적용 조건: **T ≤ 10℃ AND V ≥ 1.3m/s**
- 조건 미충족 시 기온 그대로 사용

**지역 가중 평균**: 수도권(서울 23.7% + 경기 30.2% + 인천 7.6%) 61.5% 비중 반영.

---

## Step 3. 합성 판매량 계산 (주차별)

### 3-1. 기본 판매 (월별 계절 계수)

실데이터 2025년 월별 판매 합산 ÷ 12월 합산:

```
12월 주 평균: 130,342개 (기준 = 1.00)

월별 계수 (실측):
  12월: 1.00      1월: 0.55      11월: 0.42     2월: 0.13
   3월: 0.05     10월: 0.13      4월: 0.007    5~9월: 0.001~0.007

예: 11월 첫째 주
  기본 = 130,342 × 0.42 = 54,744개
```

### 3-2. 체감온도 편차 보정

같은 달이라도 추운 해/따뜻한 해가 다름. **그 월의 2025년 평균 체감온도 대비 편차**에 구간별 민감도를 곱함:

```
계산: 편차 = 2025년 같은 달 평균 체감온도 - 2024년 이번 주 체감온도
      보정 = 편차 × 구간별 민감도 (Step 1에서 추출)

예: 2024-11-27주, 체감온도 -3℃
    2025-11 평균 체감온도 = 9.1℃ (실측)
    편차 = 9.1 - (-3) = 12.1℃ (2024가 더 추움)
    구간: -5~5℃ → 민감도 13,114/℃
    보정 = 12.1 × 13,114 = 158,679 (과도)

    → 상한 ±50% 적용 = 54,744 × 0.5 = 27,372
    → 최종 base = 54,744 + 27,372 = 82,116
```

### 3-3. 적설 효과

겨울(10~2월) 내 눈 있는 주 ÷ 없는 주 평균 = **2.29배** (실측, 겨울 내 비교로 계절 교란 제거):

```
눈 0cm: 영향 없음
눈 10cm+: × 2.29 (최대)

예: 눈 5cm → × (1 + (2.29-1) × 0.5) = × 1.645
```

### 3-4. 기온 급강하 효과

전주 대비 체감온도 **-5℃ 이상 하락**한 주 (Lee & Zheng 2024 논문 기준):

```
2025 실측 급강하 사건:
  1회차: 판매 변화 +188.4%
  2회차: 판매 변화 +102.8%
  → 시즌 내 반복 시 감쇠 (적응 효과, 논문 [B])

예: 2024-11-27 첫 급강하 주
    base × 1.884 (= 실측 1회차 배수)
```

### 3-5. SKU 분배

실데이터 2025년 SKU별 판매 비중:

```
하루온 붙이는 핫팩: 38.1%
박상병 핫팩: 12.5%
불가마 핫팩: 8.7%
... (34개 SKU)

전체 합성 판매 × SKU별 비율 = 각 SKU 행
```

### 3-6. 랜덤 노이즈

월별 실측 변동계수(CV = 표준편차 ÷ 평균):

```
12월 CV: 36.3%  → 노이즈 ±36.3%의 절반 적용
 1월 CV: 30.5%
 4월 CV: 185%   (비시즌, 변동 크지만 전체 판매 적음)

예: base 82,116 → 실제 = 82,116 ± (82,116 × 0.363 × 0.5) = 82,116 ± 14,904
```

## Step 4. 합성 납품 데이터

**월별 발주/판매 실측 비율** (납품률 xlsx × 일간성과 매칭):

```
9월: 63배     (시즌 대비 대량 선발주)
10월: 3.5배
11월: 2.4배
12월: 1.5배
1월: 0.4배    (시즌 하강)
2월: 0.16배   (거의 발주 안 함)
4~7월: 0      (사용자 정보: 8월 말부터 납품 시작)

예: 2024-12 합성 판매 = 1,037,537개
    × 1.54 = 1,597,806개 발주
```

## 이 데이터로 달성한 것

```
Before: val MAE 636 (봄만)
After:  val MAE 1,697 전체 + 겨울 MAE 2,163 첫 측정

→ "겨울 예측 정확도"라는 숫자가 처음으로 존재
→ 다음 시즌 실데이터 쌓이면 재검증 가능
```
