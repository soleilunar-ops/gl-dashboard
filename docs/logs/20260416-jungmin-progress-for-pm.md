## 3. 합성 데이터 계수 — 근거 전수 명세

### 3.1 더미 데이터 사용 여부

**합성 데이터를 제외한 모든 데이터는 실데이터입니다.**
합성 데이터(synthetic_2024_weekly.csv)는 아래 근거에 의해 생성되었으며, `synthetic=True` 플래그로 표시됩니다.

### 3.2 계수별 근거 (임의값 0건)

| 계수                   | 값                     | 근거                                                  | 태그                    |
| ---------------------- | ---------------------- | ----------------------------------------------------- | ----------------------- |
| **체감온도 공식**      | JAG/TI 2001            | 기상청 공식 채택. weatheri.co.kr + 기상청 블로그 확인 | [A] 기상청              |
| **체감온도 적용 조건** | T≤10℃, V≥1.3m/s        | 기상청 공식 적용 조건                                 | [A] 기상청              |
| **기온 구간 경계**     | 5℃ 단위                | KMITI 날씨경영 업계 관행                              | [C] 업계                |
| **한파 임계값**        | -12℃                   | 기상청 한파주의보 기준                                | [A] 기상청              |
| **기온 급강하 임계값** | -5℃                    | Lee & Zheng 2024 (JAERE) 논문 제안                    | [B] 논문                |
| **적응 효과**          | hump shape, 3주 정상화 | Lee & Zheng 2024 실증                                 | [B] 논문                |
| **월별 계절 계수**     | 12개 (12월=1.0 기준)   | 실데이터 `월 판매합 ÷ 12월 판매합`                    | [D] 실데이터            |
| **적설 배수**          | 2.29배                 | 겨울(10~2월) 내 눈 있는 주 ÷ 없는 주 평균             | [D] 실데이터            |
| **기온 구간별 민감도** | -274 ~ -13,114 /℃      | 실데이터 체감온도 구간별 평균 판매 차이               | [D] 실데이터            |
| **월별 노이즈 CV**     | 4.5%~185%              | 실데이터 `std ÷ mean` 월별                            | [D] 실데이터            |
| **지역 가중치**        | 수도권 61.5%           | 지역별판매트렌드 CSV                                  | [D] 실데이터            |
| **SKU 분배 비율**      | 34개 SKU별             | 실데이터 판매 비중                                    | [D] 실데이터            |
| **납품 비율**          | 월별 0.16~62.95        | 납품률 × 일간성과 매칭                                | [D] 실데이터            |
| **8월 납품 시작**      | ISO week 35            | 사용자 도메인 지식                                    | [E] 도메인              |
| **첫 한파 날짜**       | 2024-11-27             | 기상청 발표, Perplexity 확인                          | [E] 도메인              |
| **비시즌 범위**        | 3~9월                  | 실데이터 12월 대비 5% 미만 월                         | [D] 실데이터            |
| **체감온도 보정 상한** | ±50%                   | 과보정 방지 안전장치                                  | ⚠️ 유일한 비데이터 설정 |

### 3.3 한계

| 항목      | 내용                                                           |
