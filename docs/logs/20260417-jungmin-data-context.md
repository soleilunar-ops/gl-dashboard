## 2. 월별 계절 계수 — 데이터 기반 근거 체인

### 2.1 산출 공식

```
카테고리별 × 월별 판매합 / 카테고리별 × 12월 판매합 = 월별 계수
```

- 12월을 분모(1.00)로 정규화하는 이유: 2025-12가 데이터 내 피크 월이고, 같은 정규화 방식이 `synthetic_data_generator.py`에도 이미 적용되어 일관성 유지
- 분모를 바꿔도 발주량 계산 결과는 동일(상대 비율은 불변)

### 2.2 이상치 처리 (2026-04-17 기준 결정)

**제외 구간**: 2025-04 (데이터 시작 시점)

**근거**:

- 2025-04 붙이는 핫팩 전체 판매: **95개** (43일치 행)
- 2026-04-01~17 붙이는 핫팩: **6,534개** (17일치) — 환산 시 월 11,500개 수준
- 같은 4월인데 70배 차이 → SKU 라인업 미완(초기 6개 SKU만 등록) + 판매 정착 전 기간으로 판단
- 이 이상치를 포함하면 붙이는 핫팩 4월 계수가 0.0524 → 0.0141로 **74% 축소** (과소평가)

- **일반 핫팩 1월 2.0906** — 12월보다 1월이 2배 더 팔림. 설 선물·신년 추위 영향 추정
- **찜질팩 11월 1.1594** — 11월 피크(근육통 수요 + 초겨울 한파)

### 2.4 AI 인사이트 발주량 계산 공식

```
권장 발주량 = 기준량 × (제품 비중% / 100) × 제품의 월별 계수

기준량:
- Model B 예상 발주요청량 (model_b_category_forecast.csv pred_ratio 합)
- Model B 없으면: Model A 판매 예측 총량 × 0.6

예시 (2026-04, 붙이는 핫팩 50g, 비중 59.8%, Model B = 8,948개):
  권장 = 8,948 × 0.598 × 0.0524 = 280개 (50 단위 반올림 후)
```

## 3. 데이터의 구조적 한계 (정직한 평가)

### 3.1 단일 샘플 한계

| 한계                                     | 영향                                                                                                     |
| ---------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| 12월, 1월 등 성수기는 샘플 1개           | "그 해 그 월의 값"이지 "평균 패턴"이 아님. 2025년 특수 이벤트(프로모션·한파·경쟁사)가 계수에 그대로 섞임 |
| 분산 추정 불가                           | ±편차 범위 산출 불가 → 계수의 신뢰구간 제공 못함                                                         |
| 4월은 이상치 제외 후 2026-04 일부만 남음 | 17일치 데이터로 월 전체를 추정하는 중. 4월 후반 변동 반영 안 됨                                          |

### 3.2 SKU/카테고리 혼재

- `CATEGORY_MONTHLY_FACTOR`는 카테고리 합산이라 카테고리 내 SKU 편차 반영 못함
- 예: "붙이는 핫팩 50g" vs "붙이는 핫팩 100g" 은 크기·용도 다르지만 같은 계수 적용

### 3.3 시장 외부 요인 미반영

- 프로모션(쿠팡 딜·무료배송 조건 변경)
- 경쟁사(3M·종근당 등)의 가격/재고 변동
- 뉴스·SNS 이슈 (한파주의보 보도량 급증 시 공포 구매 효과)

## 4. 수요 예측 모델과의 관계

```
[Model A: LightGBM] ─── SKU × 주 단위 판매 예측
          │
          ├─ 피처: 기온·풍속·적설·first_snow_flag·가격·품절률 등
          └─ 학습: 합성 2024 + 실 2025-04~2025-09
             검증: 실 2025-10~2026-04 (27주, MAE 1,697 / 겨울 2,143)

[Model B: 비율 + SKU 분배] ─── 카테고리 발주량 → SKU별 분배
          │
          └─ 비율 모델: ratio_lookback_weeks=4 실측 기반

[AI 인사이트: GPT-4o-mini]
          │
          ├─ 컨텍스트: Model A 예측 + Model B 발주량 + 날씨 + 품절률 + 카테고리별 월별 계수
          └─ 출력 공식: 기준량 × 비중% × 월별계수 (제품별)
```
