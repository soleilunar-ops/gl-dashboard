## 1. 체감온도 공식 (✅ 공식 확인 완료)

### 1.1 겨울철 체감온도 공식 (기상청 공식 채택)

**출처**: JAG/TI (Joint Action Group for Temperature Indices), 2001년 8월 캐나다 토론토 회의 발표  
**채택 기관**: 한국 기상청 + 미국 NWS + 캐나다 환경부

```
W_C = 13.12 + 0.6215 × T - 11.37 × V^0.16 + 0.3965 × T × V^0.16
```

| 변수  | 의미            | 단위 |
| ----- | --------------- | ---- |
| `T`   | 기온            | °C   |
| `V`   | 지상 10m 풍속   | km/h |
| `W_C` | 체감온도 출력값 | °C   |

**적용 조건 (두 조건 모두 충족 시에만 유효)**:

- 기온 **10°C 이하**
- 풍속 **1.3 m/s (= 4.8 km/h) 이상**
- 조건 미충족 또는 산출값이 기온보다 높으면 → **기온 값과 동일하게 표출**

### 1.2 위험 단계 기준 (⚠️ 혼용 주의)

아래 두 기준이 공존하며 **혼용하면 안 됨**:

**[A] weatheri.co.kr (기상청 원본 JAG/TI 기준, 5단계)**:
| 단계 | 체감온도 범위 |
|------|-------------|
| 낮음 | 10°C ~ -10°C |
| 보통 | -10°C ~ -25°C |
| 추움 | -25°C ~ -45°C |
| 주의 | -45°C ~ -59°C |
| 위험 | -60°C 이하 |

**[B] 기상청 현행 생활기상지수 기준 (4단계)**:

- 관심 / 주의 / 경고 / 위험의 4단계로 운용되나, **구체적 임계값(-3.2, -10.5, -15.4°C)은 공공데이터포털 파일명 추론값으로 공식 출처 재확인 필요**

> 🔑 **핫팩 수요 예측 모델 적용 권고**: [A] JAG/TI 원본 기준 또는 기상청 날씨누리의 생활기상지수 API 값을 직접 호출하여 사용할 것.

### 1.3 여름철 체감온도 공식 (⚠️ 계수 재확인 권고)

기상청이 여름철에 사용하는 공식 (형태 확인, 계수는 KMA 원문 재확인 권고):

```
W_S = -0.2442 + 0.55399×Tw + 0.45535×Ta - 0.0022×Tw² + 0.00278×Tw×Ta + C
```

- `Ta`: 기온(°C), `Tw`: 습구온도(°C)
- 상수 `C`: 기존 3.5에서 개정됨 (개정 연도 및 정확한 값 → KMA 기술노트 원문 확인 필요)

## 2. 기온과 계절상품 판매의 비선형 관계

### 2.1 학술 연구: Lee & Zheng (2024) ✅

**출처**: Seunghoon Lee (University of Missouri) & Siqi Zheng (MIT)  
**논문명**: _Extreme Temperatures, Adaptation Capacity, and Household Retail Consumption_  
**데이터**: Nielsen Consumer Panel, 미국 4만~6만 가구, 2004~2019  
**발표**: JAERE (Journal of the Association of Environmental and Resource Economists), 2024-03

> ⚠️ **한국 적용 주의**: 이 논문은 **미국 데이터** 기반. 대중교통 의존도·주거 패턴이 다른 한국에 수치를 직접 대입하면 안 됨. 방향성(정성) 참고는 가능.

**주요 발견 (검증 완료)**:

1. **방문 빈도 vs 구매량 분리**:
   - 일평균 기온 10°F(-12°C) 미만 하루 추가 → 주간 점포 방문 **0.9% 감소**
   - 그러나 방문당 구매량 **1% 증가** → 총 지출은 소폭 상승 또는 중립
   - → **핫팩 모델 시사점**: 방문 감소 + 구매단가 상승이 동시 발생. 대량 비축 구매 패턴 고려 필요

2. **제품 유형별 반응 차이**:
   - 저장 가능 식품(storable food): 한파 시 지출 **급증**
   - 비저장 신선식품: 안정적 유지
   - 비식품(non-food): 한파 시 하락
   - → 핫팩은 "저장 가능 비식품"으로 일반 소비재와 다른 분류 적용 필요

3. **동태적 패턴 (적응 효과)**:
   - 점포 방문 감소는 한파 충격 **4주 후에도 1~1.5% 지속** (완전 회복 안 됨)
   - 총 구매량은 **역U자형 (hump shape)**: 충격 전주부터 비축 시작 → 3주 후 정상화
   - → **시즌 내 반복 한파 시 적응 효과 존재하나 방문 패턴은 지속 영향**

4. **모빌리티 조절 효과**:
   - 자가용 보유 → 한파 충격의 부정 효과를 최대 **50% 완화**
   - 대중교통, 라이드셰어는 완화 효과 없음
   - → **한국 도시 소비자**: 대중교통 의존도가 높아 한파 시 방문 감소 폭이 미국보다 클 가능성

### 2.2 한국 유통업계 사례 (⚠️ 일부 수치 재확인 필요)

### 3.2 적응 효과(Habituation) 검증 결과

Lee & Zheng (2024)에 따르면:

- 방문 빈도 감소: 4주 후에도 1~1.5% 지속 → **완전 적응 없음**
- 총 구매량: 충격 3주 후 정상화 → **구매 수준은 회복**
- **시사점**: 시즌 내 반복 한파에도 소비자는 방문 패턴은 변하지만 총 구매량은 회복. 핫팩 판매는 시즌 누적 기온 편차보다 **주별 이상 편차** 변수가 더 효과적인 예측 변수일 가능성

## 6. Claude Code 전달용: 검증된 수식 및 변수 정의

### 6.1 체감온도 함수 (Python)

```python
def wind_chill(T: float, V_ms: float) -> float:
    """
    한국 기상청 공식 겨울철 체감온도 계산 (JAG/TI 2001)

    Parameters:
        T     : 기온 (°C)
        V_ms  : 풍속 (m/s, 지상 10m)

    Returns:
        체감온도 (°C). 조건 미충족 시 기온 T 반환.

    출처: weatheri.co.kr / 기상청 공식 블로그 (kma_131)
    """
    V_kmh = V_ms * 3.6  # m/s → km/h 변환

    # 적용 조건: 기온 10°C 이하 AND 풍속 1.3m/s (4.8km/h) 이상
    if T > 10 or V_ms < 1.3:
        return T

    wc = (13.12
          + 0.6215 * T
          - 11.37 * (V_kmh ** 0.16)
          + 0.3965 * T * (V_kmh ** 0.16))

    # 산출값이 기온보다 높으면 기온 반환
    return wc if wc <= T else T
```
